# 松境心理微信小程序

这是松境心理官网的小程序版本。第一版为展示型小程序，包含首页、服务、团队、关于、联系五个 tab，不包含预约表、登录、支付或后台。

## 导入方式

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择本仓库的 `miniprogram/`。
4. 开发阶段可使用测试号；正式上传前将 `project.config.json` 中的 `appid` 改为真实小程序 AppID。

## 第一版功能

- 品牌与服务展示
- 咨询师团队展示
- 微信客服入口：`button open-type="contact"`
- 电话拨打：`wx.makePhoneCall`
- 微信内置地图导航：`wx.openLocation`
- 二维码展示，作为客服入口不可用时的备用联系路径

## 上线前必须检查

- 确认 `data/contact.js` 中的电话、邮箱、地址、工作时间正确。
- 确认 `data/contact.js` 中的地图坐标能在真机预览里打开到正确位置。
- 确认微信公众平台已经开通并配置小程序客服能力，否则 `open-type="contact"` 不会形成有效客服会话。
- 用真实小程序 AppID 替换 `touristappid`。
- 在 iPhone 和 Android 真机预览五个 tab、电话、地图和客服入口。
- 检查心理咨询文案，避免医疗诊断、治疗、治愈承诺类表达。

## 你需要在微信平台完成

- 公司主体注册与认证
- 小程序 AppID 获取
- 类目选择与平台审核资料提交
- 客服能力配置
- 上传、提交审核、发布
